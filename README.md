## 📖 Overview

Banglore Opticals is a premium optical shop website for a family-owned business in Coimbatore, Tamil Nadu, India. Established in 1899, it has served generations with quality eyewear and professional eye care.

This repository contains the production-ready, SEO-optimized, responsive website for Banglore Opticals.

---

## ✨ Features

- **Reusable Component Architecture** – Header, Footer, Navigation, Floating Buttons loaded dynamically
- **Modern, Premium Design** – Elegant UI with glassmorphism and subtle animations
- **Fully Responsive** – Mobile-first approach for all devices
- **SEO Optimized** – Meta tags, Open Graph, Schema markup, robots.txt, sitemap.xml
- **Performance Optimized** – Lazy loading, optimized images, modular CSS/JS
- **Accessibility** – Semantic HTML, ARIA labels, keyboard navigation
- **6 Pages** – Home, About, Products, Services, Testimonials, Contact
- **Interactive Elements** – Gallery lightbox, contact form, testimonials

## 🛠️ Technology Stack
